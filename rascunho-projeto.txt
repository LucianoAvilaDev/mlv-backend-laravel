
REGRAS DE NEGÓCIO
-----------------

- É possível navegar pelos produtos do sistema sem cadastro
- É possível adicionar produtos no carrinho sem cadastro
- Os dados do carrinho ficam armazenados nos cookies
- Para concretizar a compra é necessário ter cadastro
- Os dados do usuário são: nome, endereço, cep, cpf, telefone, email e senha
- Após concretizar a compra, a mesma é salva com os dados do produto


produto (vem da API)
-------
id: int, 
nome: string, 
descricao: string, 
categoria: string, 
imagem: string, 
preco: decimal, 
material:string, 
departamento: string

cliente
-------
id: int
nome: string
cpf: string
email: string
senha: string

item_produto
------------
id: int
produto_id: int
fornecedor: string (brazilian_provider | european_provider)
quantidade: int
valor_unitario: decimal
valor_total: decimal 

compra
------
id: int
cliente: Cliente
produtos: Produto[]
total: decimal
